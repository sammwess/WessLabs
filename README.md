# Wess Labs

Landing page pessoal — experimentos em code, IA, música, Bitcoin e cubos.

🌐 [wesslabs.com](https://wesslabs.com)

## Projetos

- **Satsboard** — Dashboard público de Bitcoin com métricas de rede, mempool e health score
- **LyraVeil** — Análise musical com IA (Ollama, OpenAI, Anthropic), tradução e TTS
- **WhisperSubs** — Gerador de legendas e transcrições com Whisper (PWA)
- **NexTranslate** — Tradução com IA local via Ollama, privada e offline
- **GymProgress** — Tracking de treinos de musculação com progressão e gráficos
- **SpeedCubeTimer** — Timer de speedcubing com Bluetooth e estatísticas
- **SpeedCubeLab** — Biblioteca interativa de algoritmos OLL/PLL com visualização 3D

## i18n

12 idiomas com detecção automática do browser e seletor manual:
PT, EN, ES, FR, DE, IT, NL, PL, SV, DA, NO, FI

## Stack

- HTML/CSS/JS puro (zero dependências)
- i18n client-side com `data-i18n` attributes
- Hospedado via GitHub Pages

## Deploy

Push para `main` → GitHub Pages publica automaticamente.
